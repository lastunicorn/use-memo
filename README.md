# React `useEffect` Demo (with State Value)

## Overview

Demo using the `useEffect` hook in React in combination with a state value that triggers the component's re-rendering.

The demo contains:

- One state value: `count`
- Two buttons to increment/decrement the `count`
- An effect that updates the document's title (see the browser's title) whenever the state value changes and triggers a component re-rendering.

Uses Vite as build tool.
